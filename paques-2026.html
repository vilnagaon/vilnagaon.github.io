Mecanique : La Chasse aux Oeufs Teatower 2026

En 2025, les oeufs etaient uniquement en ligne (3 oeufs caches sur le site = 1 code -20%). En 2026, on double la mecanique : en boutique ET en ligne.

1
Trouvez 3 oeufs sur teatower.com
3 oeufs caches sur 3 pages produit differentes. Chaque oeuf revele une lettre. Les 3 lettres forment le code promo.
2
OU trouvez l'oeuf en boutique
Un oeuf dore cache dans chaque magasin. Le trouver = -20% immediat sur votre achat. Pas besoin de code.
3
Le code donne -20%
Valable sur tout le site du 1er au 6 avril. Un seul usage par client. Cumulable avec les achats en boutique.

Pourquoi la double mecanique ?

Online : la chasse gamifiee genere du trafic sur le site + du temps passe (3 pages minimum). Le code est la recompense.
Boutique : l'oeuf cache cree un jeu pour les enfants et un pretexte de visite pour les parents. C'est du drive-to-store deguise en jeu.

L'oeuf en boutique = biais de reciprocite

Le client "trouve" l'oeuf → il se sent recompense → il achete avec -20% → il part satisfait. La mecanique de decouverte active le circuit de recompense (dopamine) bien plus qu'un simple code envoye par email.
